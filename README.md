Профессиональное Android-приложение, превращающее современный смартфон в полноценный бортовой компьютер (OBC) и систему помощи водителю (ADAS) специально для **KIA Rio (2001)**.

---

## 🌟 Основной функционал

### 🛠 Модуль А: Диагностика и Телеметрия (OBD-II & GPS)
Приложение объединяет данные из ЭБУ автомобиля (протокол **ISO 9141-2**) и встроенного GPS-модуля смартфона.

*   **Главный экран (GPS/OBD)**: 
    *   Гибридное определение скорости с возможностью выбора приоритета (OBD-II / GPS) для максимальной точности в тоннелях и городе.
*   **Расширенная телеметрия (OBD)**:
    *   `RPM`, `OBD Speed`, `Coolant Temp`, `Fuel Level`, `ECU Voltage`.
*   **Экономика и поездка**:
    *   ⛽ **Расход**: Мгновенный, средний и суммарный расход топлива.
    *   🛣 **Поездка**: Пройденная дистанция и время в пути.
*   **Особенности**: 
    *   Режим **Safe Mode** для защиты старого ЭБУ.
    *   Гибкая настройка частоты опроса PID и фильтрация шумов данных.

### 🚔 Модуль B: Сигнатурный радар (Anti-Radar)
Интегрированная система информирования о камерах ГАИ.

*   **Базы данных**: Поддержка многорегиональных баз (РФ, Европа, СНГ) с возможностью обновления по Wi-Fi/SIM.
*   **Умные уведомления**: Настраиваемая дистанция предупреждения, фильтрация типов камер (стационарные, мобильные, средний пролет).
*   **Адаптивный звук**: Изменение громкости и типа оповещений в зависимости от текущей скорости автомобиля.

### 👁 Модуль C: Интеллектуальное зрение (ADAS)
Использование нейросетей через **TensorFlow Lite** с GPU-ускорением.

*   **LDW (Lane Departure Warning)**: Контроль разметки с настраиваемым порогом угла отклонения и минимальной скоростью срабатывания.
*   **FCW (Forward Collision Warning)**: Определение объектов с выбором дистанции реакции (Короткая / Средняя / Длинная).
*   **Визуализация**: Три режима отрисовки интерфейса (Скелетный / Залитый / Минималистичный) и цветовая индикация уровня опасности.

### 📹 Модуль D: Циклический видеорегистратор (DVR)
*   **Loop Recording**: Автоматическая запись видео с настраиваемой длительностью сегментов.
*   **Событийная запись (Event Record)**: Автоматическое сохранение клипа в защищенную папку при:
    *   Срабатывании ADAS (опасное сближение/выезд из полосы).
    *   Проезде зоны Speedcam.
    *   Резком торможении (по данным G-сенсора или OBD-II).
*   **OSD (On-Screen Display)**: Наложение на видео скорости (GPS/OBD), текущего лимита зоны и координат.

## ⚙️ Система настроек
В приложении реализовано глубокое меню конфигурации:
1. **Радар**: Обновление баз, регионы, дистанции алерта.
2. **DVR**: Разрешение, FPS, управление событиями и памятью.
3. **ADAS**: Чувствительность датчиков, режимы FCW, стили интерфейса.
4. **OBD-II**: Выбор протокола, частота опроса, приоритет источника скорости.
